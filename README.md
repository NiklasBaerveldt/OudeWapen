# OudeWapen
Opas en Jan Willems wapenversameling

All weapons have been identified using chatgpt as well as some image googling for confirmation. 

If there was a mention of a sword,rifle or pistol in the papers (found in the papers/ directory) I have attempted to match it with one of the weapons in the collection. 

Feel free to message me if you want to assist in the categorization of the weapons.
